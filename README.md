# 🚗 Scroll-Driven Hero Section Animation

A modern scroll-based hero section inspired by cinematic web animations.

This project demonstrates smooth UI interactions, GSAP scroll animations, and aesthetic frontend design using React and Tailwind CSS.

## ✨ Features

- Scroll-driven car animation
- Smooth GSAP + ScrollTrigger motion
- Dynamic UI reveal on scroll
- Letter-spaced headline animation
- Modern glassmorphism stats cards
- Animated background glow and bubbles
- Responsive design

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- GSAP (ScrollTrigger)
- Vite
